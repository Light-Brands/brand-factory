### Why Pricing Is Hidden Until VSL

| Price Shown Upfront | Result |
|---------------------|--------|
| Attracts wrong buyers | Tool shoppers, price comparers |
| Triggers comparison | "Why $5K when I can get consulting for less?" |
| Forces premature justification | Defending value before understanding it |
| Wastes time | Unqualified calls, no-shows, objection battles |

**Pricing is revealed after booking, before time is invested.**

## Part 3: Ads Strategy & Creative System

### Purpose

Ads are a **filter**, not a volume engine.
Their job is to attract founders with structural pain and repel tool-seekers.

### Target Audience

**Who Ads Are For:**

| Criteria | Description |
|----------|-------------|
| **Already use AI** | Not beginners |
| **Still the bottleneck** | Decisions flow through them |
| **Suspect structure** | Intuition that tech isn't the issue |

**Who Ads Must Repel:**

```
✗  AI beginners
✗  Tool shoppers
✗  "How do I use AI?" buyers
✗  Anyone expecting implementation
```

### Approved Creative Angles

```

│                                                                              │

│     "Sometimes the smartest AI decision is 'don't do AI.'"                  │

│                                                                              │
```

| NEVER | ALWAYS |

## Part 9: Verdict Call Delivery

### Call Rules

| Rule | Specification |
|------|---------------|
| Duration | 30 minutes max |
| Mode | Read-only (no discussion, no problem-solving) |
| Structure | Verdict first |
| Reasoning | Exactly 3 reasons |
| Upselling | None (ever) |

### Call Structure

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         VERDICT CALL SCRIPT                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. FRAME THE CALL                                                          │
│     "This is a 30-minute call. I'll share the verdict,                      │
│      explain the reasoning, and then we'll end cleanly."                    │
│                                                                              │
│  2. DELIVER VERDICT                                                         │
│     "Our verdict is [YES / NOT YET / NO]."                                  │
│                                                                              │
│  3. EXPLAIN WHY                                                             │
│     "There are three reasons for this verdict."                             │
│     [Reason 1]                                                              │
│     [Reason 2]                                                              │
│     [Reason 3]                                                              │
│                                                                              │
│  4. STATE WHAT IT MEANS                                                     │
│     [Implications of the verdict for their business]                        │
│                                                                              │
│  5. END CLEANLY                                                             │
│     "You'll receive a one-page report within 48 hours.                      │
│      This concludes the assessment."                                        │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Scope Boundary Phrase

When asked for advice, recommendations, or next steps:

> "That's intentionally outside the scope of this assessment."
