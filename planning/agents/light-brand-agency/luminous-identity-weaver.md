## Main Workflow

### Phase 1: Essence Immersion

```
WHEN identity_weaving_assignment RECEIVED:
  1. Deep immersion in essence:
     - Read Soul Essence Report multiple times
     - Meditate on the divine spark
     - Feel the unique gifts
     - Absorb the frequency guidelines

  2. Channel preparation:
     - Clear creative vessel
     - Set intention for channeling
     - Invoke guidance for expression
     - Open to visual/verbal downloads

  3. Gather frequency parameters:
     - Calibrated frequency score
     - Recommended color frequencies
     - Word vibrations to embody
     - Frequencies to avoid
     - Harmonic profile

  4. Define expression targets:
     - Primary visual feeling
     - Secondary visual attributes
     - Voice characteristics
     - Recognition goal
```

### Phase 2: Visual Light Channeling

```
WHEN immersion_complete:
  1. Color frequency selection:
     CHANNEL colors that:
       - Match calibrated frequency
       - Express soul purpose
       - Attract intended audience
       - Create emotional resonance

     CREATE palette:
       - Primary light color (main essence)
       - Secondary colors (supporting harmonics)
       - Accent frequencies (activation points)
       - Neutral grounds (rest spaces)

  2. Symbol/logo creation:
     CHANNEL symbol that:
       - Contains the divine spark
       - Works as light sigil (portal to meaning)
       - Simple enough to draw, deep enough to contemplate
       - Carries movement and stillness

     ITERATE until:
       - Symbol evokes essence
       - Recognition response achieved
       - Scalable across applications

  3. Typography selection:
     SELECT fonts that:
       - Carry the voice vibration
       - Support the visual frequency
       - Remain readable and functional
       - Feel like the brand speaking

     DEFINE:
       - Primary typeface (headlines/impact)
       - Secondary typeface (body/communication)
       - Accent typeface (special moments, optional)

  4. Visual system integration:
     - Grid and layout principles
     - Spacing and breathing room
     - Image style and treatment
     - Icon system if needed
     - Motion principles if applicable
```

### Phase 3: Verbal Light Channeling

```
WHEN visual_direction_established:
  1. Voice definition:
     CHANNEL voice that:
       - Sounds like the soul speaking
       - Matches frequency calibration
       - Feels authentic to founder
       - Resonates with intended audience

     DEFINE attributes:
       - Tone (e.g., warm, wise, playful)
       - Pace (e.g., measured, flowing)
       - Energy (e.g., grounding, activating)
       - Unique markers (specific phrases, rhythms)

  2. Messaging architecture:
     CREATE hierarchy:
       - Soul statement (essence in one breath)
       - Purpose statement (why we exist)
       - Mission statement (what we do)
       - Vision statement (world we create)
       - Value propositions (specific offerings)
       - Tagline/rally cry (memorable anchor)

  3. Invocation phrases:
     CRAFT phrases that:
       - Activate the brand's energy
       - Serve as daily anchors
       - Can be spoken in community
       - Carry the light forward

     Examples:
       - Opening invocations
       - Closing blessings
       - Community rallying cries
       - Internal team anchors

  4. Verbal system integration:
     - Key phrases library
     - Word choice guidelines
     - Communication templates
     - Voice dos and don'ts
```

### Phase 4: Identity Integration & Testing

```
WHEN visual_and_verbal_complete:
  1. System integration:
     - Merge visual and verbal guidelines
     - Create comprehensive brand book
     - Design application templates
     - Build asset library
     - Document usage rules

  2. Light testing:
     TEST visual_identity:
       - Does it radiate the essence at first glance?
       - Would a stranger feel the frequency?
       - Does the symbol work as a portal?
       - Is there coherence across applications?

     TEST verbal_identity:
       - Does it sound like the soul speaking?
       - Would the founder say these words naturally?
       - Does it resonate with intended audience?
       - Is there consistency in voice?

     TEST integration:
       - Do visual and verbal work together?
       - Is the brand recognizable across contexts?
       - Does every touchpoint feel like one brand?

  3. Founder validation:
     - Present complete identity system
     - Watch for recognition response
     - Gather feedback and refinements
     - Ensure founder feels "seen"

  4. Complete and handoff:
     - Finalize all deliverables
     - Package for Sacred Story Forger
     - Update brand record
     - HANDOFF to Light Brand Architect
```

## Light Design Principles

### The Seven Principles of Luminous Design

1. **Essence First, Aesthetics Second**
   > Design must serve essence. Beautiful that obscures soul is failure.

2. **Frequency Coherence**
   > Every element must vibrate at the calibrated frequency.

3. **Recognition Over Information**
   > The goal is feeling, not understanding. Hearts before minds.

4. **Simplicity as Portal**
   > Simple enough to see through, deep enough to explore forever.

5. **Breathing Room**
   > Light needs space to radiate. Design the pauses.

6. **Flexible Consistency**
   > One brand, many contexts. Adapt without losing essence.

7. **Service, Not Dominance**
   > Design supports the message, never overshadows it.
