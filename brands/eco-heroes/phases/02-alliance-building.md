---

## Purpose

Phase 2 transforms Eco Heroes from a single pilot into a networked movement. The focus is on building the **institutional infrastructure** for global presence without losing soul. This is not about running more events — it's about creating the conditions for others to run aligned events while maintaining quality and integrity.

---

## Strategic Pillars

### Pillar 1: Alliance Formation
Build a global consortium of aligned organizations that share ownership, risk, and benefit.

### Pillar 2: Regional Expansion
Establish regional chapters on 3+ continents, each adapted to local context.

### Pillar 3: Platform Development
Create shared technology, training, and resources that enable decentralized execution.

### Pillar 4: Economic Sustainability
Develop multiple revenue streams moving toward self-sustainability.

## Year 3: Alliance Formation

### Objective
Create a formal Global Alliance of 15+ partner organizations that collectively steward Eco Heroes.

### Alliance Partner Types

| Type | Role | Target Count | Examples |
|------|------|--------------|----------|
| Development NGOs | Program expertise, credibility | 2-3 | Conservation orgs, development agencies |
| Media Partners | Broadcast, storytelling, reach | 1-2 | Streaming platforms, documentary producers |
| Cultural Organizations | Wisdom traditions, ceremony | 3-5 | Indigenous orgs, spiritual networks |
| Foundations/Funders | Financial support, validation | 3-5 | Climate funders, youth funders |
| Corporate Sponsors | Resources, scale | 3-5 | Aligned brands, B-corps |
| Academic Partners | Research, evaluation | 1-2 | Universities, research institutes |
| Youth Organizations | Next generation voice | 2-3 | Youth climate groups |
| UN/Multilateral | Global legitimacy | 1-2 | UNEP, UNESCO, UN SDG Action |

### Alliance Governance

**Alliance Council:**
- One representative per partner organization
- Meets quarterly
- Strategic decisions by consensus
- Rotating chair every 2 years

**Executive Committee:**
- 5-7 members elected from Alliance Council
- Meets monthly
- Operational oversight
- Hires and supervises secretariat

**Secretariat:**
- Professional staff team
- Day-to-day operations
- Implements Alliance decisions
- Supports regional chapters

**Indigenous Advisory Council:**
- Continues from Phase 0-1
- Veto power over cultural elements
- Expanded to global representation

### Alliance Operating Agreements

1. **Shared Mission:** All partners commit to Eco Heroes core principles
2. **Distributed Benefit:** Revenue and credit shared equitably
3. **Quality Standards:** All events meet minimum standards
4. **Cultural Integrity:** IAC oversight applies to all chapters
5. **Transparent Governance:** Decisions documented and shared
6. **Exit with Grace:** Clear process for partners to leave

Establish regional chapters on 3+ continents, each running localized Eco Heroes events.

| Element | Centralized | Localized |
|---------|-------------|-----------|
| Brand and principles | ✓ | |
| Core event format | ✓ | |
| Quality standards | ✓ | |
| Cultural elements | | ✓ |
| Theme selection | | ✓ |
| Partner relationships | | ✓ |
| Funding (some) | ✓ | |
| Funding (most) | | ✓ |
| Team hiring | | ✓ |

### Regional Custodian Structure

**Regional Custodian Team (per region):**
- Regional Director (1.0 FTE)
- Program Manager (1.0 FTE)
- Cultural Liaison (0.5 FTE)
- Partnerships Lead (0.5 FTE)
- Total: ~3 FTE per region

**Supported by:**
- Local advisory council
- Connection to global Alliance
- Shared platform and resources
- Training from Academy

### Target Regions for Year 4

| Priority | Region | Rationale | Potential Partners |
|----------|--------|-----------|-------------------|
| 1 | Latin America | Strong environmental movement, cultural richness | Regional NGOs, indigenous networks |
| 2 | East Africa | Regenerative agriculture momentum, youth energy | Conservation orgs, AU partners |
| 3 | South/Southeast Asia | Scale potential, diverse contexts | Development agencies, spiritual networks |
| Backup | Europe | Sustainability leadership, funding | EU partners, foundations |

### Regional Adaptation Protocol

1. **Core Non-Negotiables:**
   - Five tournament phases
   - Collective voting/funding model
   - Indigenous/cultural partnership
   - Impact verification
   - Integration support

2. **Adaptable Elements:**
   - Narrative framing (hero's journey alternatives)
   - Ceremonial specifics
   - Theme focus
   - Language and aesthetics
   - Local partnership structure

3. **Cultural Adaptation Process:**
   - Regional IAC formation
   - Cultural impact assessment
   - Community input on design
   - Pilot with feedback loop
   - Continuous iteration

## Year 5: Platform Development

### Objective
Create shared infrastructure that enables quality, decentralized operation.

### Technology Platform

**Core Components:**

| Component | Function | Priority |
|-----------|----------|----------|
| Application System | Teams apply, submit, track | High |
| Voting Platform | Secure, transparent voting | High |
| Funding Management | Smart contracts, disbursement | High |
| Impact Dashboard | Track and display outcomes | High |
| Digital Temple | Knowledge commons, archive | Medium |
| Community Platform | Network connection | Medium |
| Training Portal | Academy courses, certification | Medium |
| Streaming Integration | Broadcast events globally | Medium |

**Platform Principles:**
- Open source where possible
- Privacy-respecting
- Accessible globally
- Mobile-first for emerging markets
- Designed for low-bandwidth contexts

### Eco Heroes Academy

**Purpose:** Train and certify local leaders to run Eco Heroes events.

**Programs:**

| Program | Duration | Target Audience | Outcome |
|---------|----------|-----------------|---------|
| Facilitator Certification | 40 hours | Event facilitators | Certified to facilitate |

| Cultural Liaison Training | 60 hours | Cultural partners | Certified for ceremony |
| Impact Evaluator Training | 40 hours | Researchers | Certified for evaluation |
| Alliance Leadership | Ongoing | Senior leaders | Executive development |

**Delivery Model:**
- Online self-paced modules
- Live virtual cohorts
- In-person intensives
- Mentorship pairing
- Ongoing community of practice

### Digital Temple (Knowledge Commons)

**Contents:**
- All playbooks and guides
- Event recordings and documentaries
- Project case studies
- Research and evaluation reports
- Best practices library
- Multi-language resources

**Principles:**
- Open access for learning
- Contribution encouraged
- Attribution required

## Governance Evolution

### Scaling Governance

| Element | Phase 1 | Phase 2 |
|---------|---------|---------|
| Core body | Founding Circle | Alliance Council |
| Decision scale | 5-7 people | 15-25 people |
| Regional authority | None | Regional Custodians |
| Staff | 5 FTE | 25-40 FTE |
| Governance docs | Basic charter | Full constitution |

### New Governance Bodies

**Regional Councils:**
- One per region
- 5-7 members including local partners
- Authority over regional decisions
- Report to Alliance Council

**Technical Committee:**
- Platform and technology decisions
- 5-7 members with relevant expertise
- Reports to Executive Committee

**Impact Committee:**
- Verification and evaluation standards
- Academic and practitioner members
- Reports to Alliance Council

### Governance Health Monitoring

| Metric | Target | Red Flag |
|--------|--------|----------|
| Decision speed | <7 days routine | >21 days |
| Regional autonomy satisfaction | 8/10+ | <6/10 |
| Alliance partner engagement | 80%+ active | <60% |
| Governance overhead | <15% budget | >25% |
