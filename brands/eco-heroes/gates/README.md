## Failure Modes

If a gate cannot be completed:

1. **Identify blockers:** What specifically is preventing completion?
2. **Escalate to Council:** Report to full Council for guidance
3. **Assess options:**
   - Can requirements be modified while preserving intent?
   - Is more time needed?
   - Is the project fundamentally unviable?
4. **Council decision:** Proceed, pause, or terminate

**The Council reserves the right to halt the project if gates cannot be completed in good faith.**
