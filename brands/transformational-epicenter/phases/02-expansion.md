# Phase 2: Expansion

> *From one Epicenter to many—scaling transformation responsibly.*

**Status:** Future
**Timeline:** Year 3-5+
**Focus:** Replicate, scale, and build toward 30+ global locations

---

## Phase Objectives

1. Achieve operational excellence at Mexico location
2. Develop replication playbook and systems
3. Identify and secure second location
4. Build leadership pipeline
5. Strengthen brand and market position
6. Establish training academy
7. Create sustainable growth engine

---

## Prerequisites for Expansion

Before opening a second location:

| Criterion | Requirement | Measurement |
|-----------|-------------|-------------|
| **Operational Excellence** | Mexico running smoothly | Guest satisfaction >4.8/5, zero serious incidents |
| **Team Depth** | Leadership for new location | 2+ leaders ready to deploy |
| **Financial Sustainability** | Proven unit economics | Mexico profitable or clear path |
| **Demand Clarity** | Demonstrated demand | Wait list, referral volume |
| **Playbook Complete** | Replicable systems | Documented, tested |
| **Capital Access** | Funding for expansion | Committed or accessible |

## Replication Playbook

### Playbook Components

| Component | Description |
|-----------|-------------|
| **Site Selection Guide** | Criteria, evaluation process, due diligence |
| **Facility Standards** | Design guidelines, minimum requirements |
| **Operational Protocols** | All medical, clinical, hospitality procedures |
| **Training Curriculum** | Complete programs for all roles |

| **Legal Templates** | Entity structure, contracts, compliance |
| **Quality Assurance** | Audit frameworks, continuous improvement |

### Standardization vs. Localization

| Standardized (Non-Negotiable) | Localized (Adaptable) |
|------------------------------|----------------------|
| Medical protocols | Architectural style |
| Safety standards | Local materials and suppliers |
| Ceremony framework | Cultural adaptations |
| Guest journey stages | Food and nutrition specifics |
| Technology platform | Staff recruitment channels |
| Brand identity | Community partnerships |
| Quality metrics | Regulatory compliance specifics |
| Training core | Language and communication |

## Risk Management

### Expansion Risks

| Risk | Mitigation |
|------|------------|
| **Quality dilution** | Rigorous standards, training, audits |
| **Leadership shortage** | Pipeline development, academy |
| **Regulatory changes** | Diversified locations, legal monitoring |
| **Brand damage** | Standards enforcement, incident protocols |
| **Over-extension** | Deliberate pace, prerequisites |
| **Cultural misalignment** | Values-first hiring, onboarding |

### Growth Principles

1. **Quality before quantity:** Never sacrifice standards for speed
2. **Prove before replicate:** Each model must be proven before copying
3. **Leadership first:** Don't open without ready leadership
4. **Demand-driven:** Open where demand is clear
5. **Sustainable pace:** 2-4 locations/year maximum
