## Brand Essence

### Visual Principles

| Principle | Expression |
|-----------|------------|
| **Grounded Transcendence** | Earth tones with ethereal accents—rooted yet reaching |
| **Medical Trust** | Clean lines, professional polish, clarity |
| **Luxury Without Excess** | Refined elegance, not ostentatious display |
| **Sacred Containment** | Warmth, safety, intentional space |
| **Natural Integration** | Organic shapes, biophilic textures |

### Design Pillars

1. **Trustworthy:** Every visual choice reinforces safety and professionalism
2. **Warm:** Inviting, never cold or clinical
3. **Spacious:** Generous white space, room to breathe
4. **Intentional:** Every element serves purpose
5. **Timeless:** Classic elegance over fleeting trends

## Spacing & Layout

### Spacing Scale (8px Base)

| Token | Value | Usage |
|-------|-------|-------|
| `space-1` | 4px | Tight spacing, icon padding |
| `space-2` | 8px | Small gaps, inline spacing |
| `space-3` | 12px | Compact elements |
| `space-4` | 16px | Standard element spacing |
| `space-5` | 24px | Component padding |
| `space-6` | 32px | Section separation |
| `space-7` | 48px | Large section gaps |
| `space-8` | 64px | Major section divisions |
| `space-9` | 96px | Hero/full-page sections |
| `space-10` | 128px | Maximum spacing |

### Grid System

| Breakpoint | Columns | Gutter | Margin |
|------------|---------|--------|--------|
| **Mobile** (<640px) | 4 | 16px | 20px |
| **Tablet** (640-1024px) | 8 | 24px | 40px |
| **Desktop** (1024-1440px) | 12 | 32px | 80px |
| **Wide** (>1440px) | 12 | 32px | auto (max 1280px content) |

### Layout Principles

1. **Generous White Space:** Never crowd elements; let content breathe
2. **Asymmetric Balance:** Interest through intentional imbalance
3. **Content Width:** Body text max 65-75 characters per line
4. **Vertical Rhythm:** Consistent spacing creates visual calm

## Photography & Imagery

### Photography Style

| Element | Direction |
|---------|-----------|
| **Lighting** | Soft, natural light; golden hour preferred |
| **Color Grading** | Warm, slightly desaturated, earth tones enhanced |
| **Composition** | Spacious, contemplative, rule of thirds |
| **Subjects** | Real people in genuine moments, nature, architecture |
| **Mood** | Calm, grounded, hopeful, intimate |

### Photography Categories

| Category | Usage | Direction |
|----------|-------|-----------|
| **People** | Guests, team, testimonials | Authentic, diverse, un-posed, showing genuine emotion |
| **Nature** | Backgrounds, section breaks | Mexican landscape, gardens, water, sky |
| **Architecture** | Facility showcase | Low-density, integrated with nature, warm materials |
| **Details** | Modalities, food, elements | Close-ups of treatments, meals, botanical details |
| **Sacred** | Ceremony preparation | Respectful, abstract, never exploitative |

### Image Treatment

```css
/* Standard Image Treatment */
.image {
  filter: contrast(0.95) saturate(0.9);
  border-radius: 8px;
}

/* Hero Image Overlay */
.hero-overlay {
  background: linear-gradient(
    180deg,
    rgba(42, 37, 32, 0) 0%,
    rgba(42, 37, 32, 0.6) 100%
  );
}
```

### What to Avoid

- Stock photography feeling
- Overly posed or artificial
- Cold, clinical environments
- Exploitative ceremony imagery
- Over-saturated or heavily filtered
- Generic wellness clichés

## Motion & Animation

### Motion Principles

| Principle | Application |
|-----------|-------------|
| **Purposeful** | Motion guides attention, never distracts |
| **Organic** | Easing curves inspired by nature |
| **Calm** | Slow, measured transitions |
| **Respectful** | Reduced motion option always available |

### Animation Tokens

| Token | Duration | Easing | Usage |
|-------|----------|--------|-------|
| `motion-subtle` | 150ms | ease-out | Hover states, micro-interactions |
| `motion-standard` | 250ms | ease-in-out | Page transitions, reveals |
| `motion-emphasis` | 400ms | cubic-bezier(0.16, 1, 0.3, 1) | Important state changes |
| `motion-dramatic` | 600ms | cubic-bezier(0.16, 1, 0.3, 1) | Full-page transitions, modals |

### Common Animations

```css
/* Fade In Up - Content reveal */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Gentle Pulse - Subtle attention */
@keyframes gentlePulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.7; }
}

/* Scale In - Modal/card entrance */
@keyframes scaleIn {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}
```

## Logo & Wordmark

### Primary Logo

The Transformational Epicenter logo combines:
- **Wordmark:** "Transformational Epicenter" in Cormorant Garamond
- **Symbol:** A stylized spiral/seed form suggesting transformation and grounding

### Logo Versions

| Version | Usage |
|---------|-------|
| **Full Logo** | Primary applications, adequate space |
| **Wordmark Only** | Horizontal constraints, formal contexts |
| **Symbol Only** | App icon, favicon, small applications |
| **Stacked** | Square format requirements |

### Logo Colors

| Context | Primary | Secondary |
|---------|---------|-----------|
| **Light Background** | Epicenter Teal (#1A4D4D) | Deep Earth (#2A2520) |
| **Dark Background** | Warm White (#FAF8F5) | Sacred Gold (#C9A962) |

### Clear Space

Minimum clear space = height of the "T" in Transformational

### Minimum Sizes

| Version | Print | Digital |
|---------|-------|---------|
| **Full Logo** | 1.5 inches wide | 150px wide |
| **Wordmark** | 1 inch wide | 100px wide |
| **Symbol** | 0.5 inch | 32px |

## Application Examples

- **Background:** Warm White (#FAF8F5)

- **Progress Indicators:** Sacred Gold

- **Paper:** Uncoated warm white stock
- **Ink:** Deep Earth for text, Epicenter Teal for accents
- **Finish:** Soft touch coating, gold foil accents sparingly

- **Materials:** Natural wood, stone, brushed metals

- **Lighting:** Warm LED, never harsh
