# Chrono-Commune Platform Overview

> Comprehensive analysis of collaborative calendar platform architecture and feature requirements

---

## Platform Vision

Chrono-Commune is a collaborative calendar and communication platform designed to make schedule sharing and coordination incredibly easy. Available as a mobile app (iOS and Android) and web application, the platform serves as a comprehensive scheduling and communication solution.

---

## Core Purpose

Chrono-Commune serves as an all-in-one solution for:

- **Calendar Sharing** - Between family, friends, colleagues, and various communities
- **Event Coordination** - With real-time notifications and updates
- **Group Communication** - Tied directly to calendar events
- **Unified Experience** - Eliminating the need for multiple apps by combining scheduling, messaging, and photo sharing in one platform

---

## Key Features

### 1. Calendar Management

#### Multiple Calendar Support
- Create separate calendars for different groups (family, work, friends, hobbies)
- View multiple calendars simultaneously
- Switch between calendars easily
- Rearrange calendar list order
- Configurable calendar and member limits

#### Calendar Views
- **Monthly View** - Traditional grid layout with all dates visible
- **Weekly View** - Vertical hourly timeline showing 7 days
- Switch views instantly with toggle buttons
- Today button for quick navigation
- Month/year navigation controls

#### Calendar Settings
- Customizable calendar name and cover image
- Calendar Profile toggle
- Location information in photos (can be enabled/disabled for album uploads)
- Event Label Management
- Public Holiday settings (can show/hide, import from external sources)
- Ability to delete calendars

---

### 2. Event Creation & Management

#### Basic Event Features
- Title entry
- Start and end date/time selection
- All-day event option
- Event color/label customization (10 label colors available)
- Participant selection from calendar members

#### Advanced Event Options
- **Repeating Events** - Set recurring schedules
- **Notifications** - Customizable reminders
- **URL Attachment** - Link relevant websites
- **Location** - Add addresses or place names
- **File Attachments** - Attach documents and files
- **Notes** - Add detailed descriptions
- **To-Do Lists** - Create task lists within events
- **Save as Memo** - Convert events to memo format
- **Time Zones** - Set events in different time zones

#### Event Management
- Copy, edit, and delete events
- Search functionality across entries
- Event customization with labels and colors
- Event priority (Premium feature)

---

### 3. Collaboration Features

#### Member Management
- Invite members via sharing links or invitations
- Member roles: Creator/Admin and regular members
- Member visibility: See who's in the calendar
- Last online status: Check when members were last active
- Member deletion: Only admins can remove members
- Profile management: Each member has a profile with avatar

#### Notifications System
- Automatic notifications when events are created or modified
- "Once a day" notification option to reduce frequency
- "Notify me only of events I'm participating" option
- Stop notifications per calendar
- Real-time updates keep everyone informed

#### Calendar Sharing
- Share entire calendars with others
- Control who can view and edit
- Show/hide specific calendars to members
- "Show only this calendar" quick filter

---

### 4. Communication Tools

#### Memo Feature
- Text-based notes for each calendar
- Quick memo input with "Press Enter to input"
- Shared across all calendar members
- Easily accessible from right sidebar

#### Activity Feed
- Tracks all calendar activities
- Separate tabs for Events and Album activities
- Shows recent changes and updates
- Helps members stay informed about calendar changes

#### Chat/Comments
- Discussion tied to specific events
- No need for separate messaging apps
- Contextual communication

---

### 5. Album Feature

#### Photo Sharing
- Dedicated album for each calendar
- Upload and share photos with calendar members
- Organize memories (family moments, children's growth, trip memories)
- Add photos button for easy uploads
- Location information can be included/excluded
- Activity tracking for album updates
- Available on web version

---

### 6. Integration & External Calendars

#### Import Capabilities
- Import external calendar events
- Sync with Google Calendar
- Share external calendars within platform
- View external calendar events within the app
- Troubleshooting support for sync issues

#### Limitations to Address
- Export functionality required
- Printing options needed

- Email registration with verification

#### User Settings

## Summary

Chrono-Commune represents a sophisticated yet user-friendly calendar sharing platform that goes beyond simple scheduling. It combines calendar management, event coordination, communication, photo sharing, and collaboration tools into one cohesive ecosystem.

The platform eliminates the need for juggling multiple apps by providing everything needed for group coordination in one place. With a free tier offering robust functionality and a premium tier for power users, the platform can serve individuals, families, friend groups, and professional teams globally.
