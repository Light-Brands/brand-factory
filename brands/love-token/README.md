## Vision

Love Token is building the world's first **Love Economy** — an ecosystem where:

- **Meditation IS contribution** — not a reward, but the source of value
- **Communities participate as equals** — never recipients, always co-creators
- **Love flows in circles** — donors receive love back, meditators earn real value
- **The sacred is practical** — grounded spirituality, not abstract idealism

### The Three Pillars

**Grounded Love** — Love that manifests materially. Credits that buy groceries, pay for services, circulate through local economies.

**Spiritual Love** — Love cultivated through presence and intention. Meditation as sacred practice generating outer value.

**Living Love** — Love in motion. Continuous flow between donors, meditators, communities, and vendors.

## Development

### Application Directory

All Love Token application code will be developed at:

```
/brands/love-token/app/
```

### Design System

Love Token uses **design-system-v2** (Onyx Design System):

```
/design-system/design-system-v2/
├── components/        # 50+ React components
├── tokens/           # Design tokens (customize for Love Token)
└── spec/             # Brand-agnostic templates
```

### AI Coding Configuration

Development leverages **ai-coding-config**:

```
/design-system/ai-coding-config/
├── rules/            # Coding standards
├── .claude/commands/ # Workflow automation
└── .claude/agents/   # Specialized AI assistants
```
